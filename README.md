# Prometheus based monitoring



## Состав:


* Prometheus
* alertmanager
* karma
* node_exporter


## Механика подъема


docker-compose up -d 

## Что работает

* http://0.0.0.0:9090 -- prometheus
* http://0.0.0.0:9093 -- alertamanager
* http://0.0.0.0:3000 -- grafana
* http://0.0.0.0:8080 -- karma
